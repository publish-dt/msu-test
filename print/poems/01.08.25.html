01.08.2025 Катрен №01 (3364) “Выстроен Новый Мир, как Небо!”

Подошло время, Россия начала формироваться!
Главное, Ей начинать с Первого человека!
Россия, по этому Принципу, начала строиться!
Теперь, в Вечности, или, на другие века!

Если есть в Духе, Лидер, значит, есть страна!
Главное, для страны, правильное направление!
Сегодня, для Мира людей, война, как страда,
Россия, ускорила, вверх страны движение!

Лидер, конечно, очень важен, как пример!
Он от Создателя, поэтому для людей, икона!
Никаких, не должно быть для Равенства мер,
Не должна для Лидера быть, на голове Корона!

Это было в прошлом, теперь, другое в настоящем!
Человек не может заглянуть в Завтра!
Идеология – это, для Завтра, самое стоящее!
Лидер, идущий впереди, несёт Правду!

Россия, уже разбита, Судьбой, на фракталы!
Восемь, для Мира сего, роковое число!
Россия, открыла в Небе, Свои Порталы,
Объединённые Богом, уже, в Одно!

Россия уже Пирамида, с Вершиной в Небе!
Каждый уровень – это свой Этап!
Для России, Шестая Эпоха, как Небыль!
Россия, это к Создателю трап!

Создатель начал процесс, чистки!
Начиная, с Сознания каждого человека!
Для этого, не должно быть, никакой вывески,
Человек, должен быть готов, на века!

Создатель повторяет, что жизнь вечна!
Однако, форма проявления, разная!
Нельзя, человеку жить беспечно,
Жизнь, не может быть праздной!

Шестая Эпоха, ведёт людей ввысь!
Материя осталась в Пятой Эпохе,
Не может быть на Небе, корысть,
До Начала Перехода, остались, времени крохи!

Теперь, для людей есть готовый пример!
Пример в Духе и, ещё, в Материи!
У каждого человека в Мире, есть номер!
В Пятой Эпохе, номера были потеряны!

Создатель, теперь, говорит о Новом!
О Новом поле и о состоянии человека!
Люди, должны вместе, сойтись в этом!
В построении, совершенно, Нового Века!

Создатель, уже, утвердил Мессию!
Значит, появилась Точка Отсчёта!
Значит, началась Переходная Сессия,
Другими словами, Бог объявил Начало Полёта!

Все разговоры о войне, закончились!
Война не перейдёт в Шестую Эпоху!
Задачи России, уже, определились!
Россия, в Новый Мир, перейдёт сходу!

Создатель, подтвердил Мира готовность!
Ибо, три уровня Пирамиды определены!
Россия, должна проявить активность,
Первые лица, Октаэдра, выделены!

Идёт формирование страны, внутри!
Продолжается, чистка рядов!
Старое управление страной, должно уйти!
Не должна, расти, гора гробов!

Всё будет по Системе Бога и Его управления!
Равенство всех частиц – Его аргумент!
Должно быть к Одной Цели, движение!
Россия – это Создателю Монумент!

Выстроен Мир, как Небо!
Впереди Бог, а за ним, для людей, Мессия!
Вы, с Создателем, на грехи наложили Вето!
Начинается Нового Мира, Сессия!

Выстраивается Пирамида в Духе!
Есть Энергетическое обрамление!
В России, азарт Нового, не потух,
Вас, ещё ждут, Новые достижения!

Да будет Свет Истины!
Отец Абсолют или Высший Космический Разум!
