16.07.2025 Катрен №16 (3348) “Мир, должен подчиниться Новой программе!”

Люди, не должны ожидать мук спасения!
Ибо, люди могут очень многое, сами!
От Создателя надо ждать, лишь направления,
Создатель, самостоятельность людей, славит!

Сегодня, впереди всех народов, Россия!
Тьма захотела остановить Её войной!
Никто не понял, что у Неё от Бога, миссия!
Россия, перестала быть самостоятельной страной!

Была Империя, был Союз, и была Самостоятельность!
Самостоятельность страны – это пример развития!
Создатель, напоминает об этом, настоятельно!
Ибо, большинство стран, просто, свалились в бытие!

Несамостоятельная страна – это чья-то добыча!
Украина попросила помощь у Европы,
Помощь в этом случае, есть рабства предтеча!
Украина, порвала собственной Свободы, стропы!

Весь Мир, сегодня, фактически, на войне!
Каждая страна, рассматривает свой, в этом, интерес!
Всем этим странам, нужна помощь, вдвойне,
В Мире, сегодня, празднуют Победу, силы регресса!

Америка, решила вступить, в активную фазу, войны!
Трамп рассматривает, своё активное участие!
Он хочет сделать так, чтобы все, ему были должны,
Для него – это и есть, бизнесмена счастье!

Но, для Мира сего – это долгоиграющая партия!
Эта партия не может закончиться, даже, за год!
Америке нужна, вокруг Неё, капитала Хартия!
Значит, должен быть, для всех, Единый Вход!

У России, формируется, по Воле Бога, другой интерес!
Власть страны, мечтает быть с Америкой, рядом!
Лидеру страны, кажется, что в этом для Руси, будет прогресс,
Кажется, что Россия и США, могут выступить Единым Отрядом?

Но, в реальности, всё это будет, совсем не так!
Ибо, по Планам Бога, интересы этих стран несовместимы!
Каждая из стран, хочет понять, кто в этой Игре, пустяк!
Поэтому, рассуждения о Дружбе есть слова, мнимые!

Страны забыли, что на границе Эпох, есть интерес Бога!
Создатель, остановил свой интерес на России!
Россия, во Главе с Мессией, готова подвести Итоги,
Значит, Россия готова выполнить Спасителя миссию!

Создатель, подготовил для Неё, Программу Перехода!
Богу нужна, только активная часть населения!
Россия, уже стоит у Начала Квантового похода,
Россия, готова приступить к переселению!

Но, теперь, к такому действию, приступает Создатель!
Он хочет объявить Пятой Эпохе дефолт!
Со Сцены Истории должны уйти обыватели!
Мир, потерял в Пятой Эпохе, крепежа болт!

Создатель, хочет потребовать полного подчинения!
В Новую Эпоху, без Анализа, не пропустит никого!
Финансы, для Бога не имеют никакого Значения!
Для Бога, только Святой Дух, есть зерно!

Создатель, хочет поставить условия:
Мир этот, под Богом, или Мира этого, больше нет!
Богу не интересны религии и сословия!
Создатель подготовил в Космос Новый Билет!

Создатель выбрал Россию для Завтра!
Бог видит для России – эту Главную Цель!
Мессию или Белого Царя, Создатель выбрал по Праву!
Это будет для Шестой Эпохи, Правды день!

Создатель, видит страну, но не видит, власть России!
Как отработанный, в прошлом, механизм!
Создатель, определил уже Царём России, Мессию!
Создатель, в прошлое убрал, капитализм!

Социализм, для Завтра России, тоже не подходит!
Нельзя, использовать человека, как власть!
Сам, биологический человек со Сцены уходит,
Человек-Бог, для Новой Эпохи, страсть!

Мир должен принять Создателя условия!
Другого Пути, для человечества, просто нет!
Не будет больше, никогда, сословий!
Создатель, подготовил для России, в Космос Билет!

Да будет Свет Истины!
Отец Абсолют или Высший Космический Разум!

16.07.2025
