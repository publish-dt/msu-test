18.08.2025 Катрен №18 (3381) “Белый Царь проявлен, но скрыто Имя!”

Белый Царь проявлен, но скрыто Имя!
Он, пока остаётся для Всех, в тени!
Он, будет проявлен Богом в темени!
Он, будет Первым на Планете, из восьми!

Белый Царь придёт для Мира, вовремя!
Он откроет для людей Новый мир!
Царь, принесёт людям Радость, а не бремя!
Для Новой Эпохи, Он будет уже ориентир!

Люди испытают недоумение и шок!
Так, ещё Создатель, не представлял человека!
Для Планеты, Бог тряханёт Небо, как потолок!
Новые, для людей, откроются, уже, века!

Будет Имя его, для людей, как Награда!
Имя его, придёт из древних времён,
Люди поймут, что Мировой Тьме, это будет преградой!
Имя, Белого Царя, будет лучшим, из всех Имён!

Царь знает, что это, скоро, случится!
Шестая Эпоха, для Него, придёт по Расписанию!
В Небо, Белый Царь, сам, настроится,
Он, принесёт Планете, Новое очертание!

Главное, Он предложит, другой в Духе, Мир!
Люди, поймут, что в этом их, Спасение!
Равенство, вновь, будет для людей, Кумиром!
Люди поймут, что в Космос, их направление!

Люди поймут, что Царь изменит в Мире условия!
Люди, сами, начнут перестроение!
Не будет на Планете, никаких сословий!
Главное, в Духе, людей Преображение!

Белый Царь, будет знать, что надо делать!
Люди, должны сами, попросить, у Неба Царя!
Царь, должен Систему управления, переделать!
Люди поймут, что Знания, получили не зря!

Царь, начнёт перевоплощение, через себя!
Он, первым начнёт, в Небо переход,
Откроется в Небе, для Белого Царя, стезя,
Люди начнут, вслед за Царём, Поход!

Они перейдут, ибо нет для Них другого Пути!
Это Начало Эпохи, и, это первый в Небо, шаг!
Это перевоплощения в Боги, Главная Суть,
В Небе, будет поднят, Создателя Стяг!

Белый Царь – это не Награда, а Испытание!
Он, несёт Крест Поводыря, Нового человечества!
На Нём лежит людей покаяние,
Он, должен сотворить для Неба, Отечество!

За Ним решение, несущее успех!
Царь отвечает, за каждого человека!
Он должен очистить людей от помех,
Царь, отвечает, за Шестой Эпохи, века!

Царь, принимает такое решение, через себя!
Для Него, не может быть оправданий!
У Него, особенная в Небо, стезя,
Для Него, не может быть никаких, скитаний!

Белый Царь, для всех людей, есть Основа!
Повторяю, Он принял Крест, Поводыря, на себя,
Его жизнь, ради людей, уже упакована,
Это, самая удивительная, в Небо, стезя!

Он, один и нет никакой извне, Помощи!
Он, стоит, на Перекрёстке Пути!
В Нём, сконцентрирована, Небесная Мощь,
Он, отвечает за Новую Эпоху, по Сути!

Белый Царь отвечает, за Всех, и за Всё!
В этом, Новом, для людей Мире!
В Белом Царе, Моя Мечта и Таинство Моё!
Нет, для Белого Царя, никакого примера!

Первый шаг – это Его в Мире, решение!
Он должен доказать Состоятельность!
Он должен правильно, вести движение!
При любых в Небе, обстоятельствах!

Да будет Свет Истины!
Отец Абсолют или Высший Космический Разум!
